Welcome to EarthWeather!

Welcome to the EarthWeather! This app is used for monitoring data sensors and forecasts weather. Helps you in cultivating ornamental plants and make them more effective and efficient in cultivation.

Data Sensors

Temperature

Sensor DHT11 digunakan untuk mengukur suhu lingkungan.

Data Temperature didapatkan dari sensor DHT11 ini.

Humidity

Sensor DHT11 juga digunakan untuk mengukur kelembaban udara sekitar.

Data humidity atau kelembaban didapatkan dari sensor DHT11 ini.

Soil Moisture

Sensor Soil Moisture digunakan untuk mengukur kelembaban tanah.

Data moisture didapatkan dari sensor Soil Moisture ini.

Ultrasoinic HCSR04

Sensor Ultrasoinic digunakan untuk mengukur tinggi wadah air.

Data jarak ketinggian air didapatkan dari sensor Ultrasoinic ini.

Data Forecasts Weather

Cuaca

Temperature °C

Berisi data Temperature dalam celcius dari cuaca.

Temperature °F

Berisi data Temperature dalam fahrenheit dari cuaca.

Kode Cuaca

Humidity
